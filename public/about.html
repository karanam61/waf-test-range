About Us

WAF Test Range is a fictional corporate portal maintained by the internal security engineering team. It simulates a typical small business web application with catalog search, user accounts, file sharing, and feedback forms.

This site exists solely to generate realistic HTTP traffic patterns for Web Application Firewall policy validation. It is not a production service and must always run behind a WAF in a restricted network.
